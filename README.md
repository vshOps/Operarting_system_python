# CPU Scheduling Algorithms

This project demonstrates the implementation of various CPU Scheduling Algorithms in Python. It provides a comparative understanding of Preemptive and Non-Preemptive scheduling techniques used by operating systems to manage process execution efficiently.

## Overview

CPU Scheduling is a fundamental concept in Operating Systems that determines which process gets access to the CPU and for how long. This project simulates different scheduling algorithms and calculates performance metrics such as Waiting Time, Turnaround Time, and Completion Time.

## Features

- Implementation of Preemptive Scheduling Algorithms.
- Implementation of Non-Preemptive Scheduling Algorithms.
- Automatic calculation of:
  - Completion Time (CT)
  - Turnaround Time (TAT)
  - Waiting Time (WT)
- User-friendly console-based interface.
- Comparative analysis of scheduling methods.

## Scheduling Algorithms Implemented

### Non-Preemptive Algorithms
- First Come First Serve (FCFS)
- Shortest Job First (SJF)
- Priority Scheduling


### Preemptive Algorithms
- Shortest Remaining Time First (SRTF)
- Priority Scheduling (Preemptive)
- Round Robin (RR)
